#Sistema de controle financeiro

Sistema desenvolvido em python para análise empresarial utilizando Pandas.

#Funcionalidades:
 - Faturamento Total
 - Faturamento por Loja
 - Faturamento Loja/Produto
 - Faturamento e Quantidade Vendida por Produto
 - Menu Interativo no Terminal

#Tecnologias Utilizadas:

 - Python
 - Pandas
 - Excel

#Como Executar:

1. Instale as Dependências: pip install pandas openpyxl
2. Execute o Programa: "empresax.py"
